# Tampermonkey UserScript
(1) Analog resizeable svg clock, controlContainer, and dateTimeContainer are all draggable.

(2) Options to change wallpaper (0 - 52), manually or auto change hourly, daily, weekly, monthly.

(3) Option to toggle wallpaper host site between GitHub, ImgBB, or Postimages.

(4) Display custom logos (0 - 17), default Google logo or leave blank.

(5) Clock face light and dark themes.

(6) Second hand movement options between ticking or smoothe sweeping.

(7) Logo images are hard coded into script. For less scrolling, go into Settings, scroll down to Editor and uncheck Word wrap, then Save.
